🍔 FoodPanda_Sales_Analysis_CaseStudy

Project Overview

This project analyzes 2 years of Foodpanda Pakistan order data using Microsoft Excel. The objective was to uncover business insights related to revenue generation, customer behavior, restaurant performance, delivery efficiency, and customer churn.
Using data cleaning techniques, Pivot Tables, Pivot Charts, and an interactive dashboard, the project transforms raw operational data into actionable business insights that can support decision-making.

Business Objectives

The analysis aims to answer some questions like:

* Which cities generate the highest revenue?
* Which restaurants contribute the most sales?
* Which restaurants have the highest customer volume?
* Is there a relationship between late deliveries and customer churn?
